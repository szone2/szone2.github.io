运算符（上）

算数运算符

定义 a = 10 , b = 20

操作符 描述 例子
+ 加法 - 对操作符的两侧增加值 a + b = 30
- 减法 - 左侧操作数减去右侧操作数 a - b = -10
* 乘法 - 运算符两侧的值相乘 a * b = 200
/ 除 - 由右侧操作数除以左侧操作数 b / a = 20
% 模 - 由右侧操作数除以左侧操作数返回余 b % a = 0
** 指数 - 执行对操作数（幂）的运算 a ** b = 10
// 地板除 - 操作数的除法，其中结果是将小数点后的为数被除去的商 9 // 2 = 4 而 9.0 // 2.0 = 4.0
