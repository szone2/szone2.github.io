PyCharm常用快捷键

ctrl + c ：快速复制一行

ctrl + d ：快速复制选中内容

ctrl + shift + n ：找查项目中的文件

ctrl + alt + l ：自动调整好代码格式

alt + enter ：查找所需的包，选中后直接import进来

ctrl + / ：注释选中的内容

shift + tab ：往左移动一个tab的距离

shift + enter ：跳到下一行

ctrl + enter ：跳到上一行
